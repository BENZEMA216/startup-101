### 石头《给年轻 AI 创业者的融资指南》⭐ 镇场之作

- **作者/来源：** 石头看未来（锦秋基金投资副总裁，36氪 / 极客公园 / 变量资本前背景，11 年一级市场经验）
- **发布：** 2025-05-04
- **篇幅：** 3.5 万字
- **URL：** https://mp.weixin.qq.com/s/T7IdFuJRiyhdOYeVaEKfOQ
- **本地 clip（自用，开源版删除）：** `$OBSIDIAN_CLIPPINGS/给年轻AI创业者的融资指南：3.5万字拆解从0到1的策略、实战与避坑.md`
- **适用阶段：** Stage 6 Fundraising 全程（本 skill 该阶段的主干参考）
- **一句话价值：** 目前中文世界最完整的 AI 创业融资 cookbook，锦秋投资副总裁亲笔，实战感极强

**文章结构（17 节目录，skill 用户按 profile 跳读）：**

#### 第一部分：融资基础与决策（§1-3）
- §1 理解融资交易：你在卖什么，投资人买什么
- §2 为什么要融资
- §3 换位思考：投资人如何退出（IPO 难度 / 并购 13-16 倍 PE / 创始人真实回报）

#### 第二部分：融资实战（§4-10）
- §4 梳理资本故事（对标叙事 / 交个朋友 / 卡脖子 / 水电煤）
- §5 BP 撰写（不同轮次不同侧重点）
- §6 Pitch 投资人（中美知名 AI VC 列表）
- §7 FA 的选择
- §8 管理融资进程（哄抢效应 / 持续融资能力）
- §9 融资过程拆解（TS → DD → IC → SPA → Closing）
- **§10 条款避坑（五大类核心条款 + 回购 + 对赌）⭐ 最重要**

#### 第三部分：融资后（§11-13）
- §11 出海架构（VIE / 红筹 / 开曼三明治）
- §12 投后关系管理
- §13 融资宣传

#### 进阶篇（§14-17）
- §14 融资宣传策略
- §15 战略布局
- §16 非明星创始人的融资策略
- §17 AI 细分赛道估值差异

**核心要点（Skill 按阶段触发）：**

1. **§1.5 早期估值不重要** — 合理范围即可，重心是融到钱 + 找对人 + 快跑
2. **§6.4 海王心态** — 广撒网 / 不依赖 / 控节奏
3. **§8.1 哄抢效应** — 节奏制造是 founder 职责
4. **§10.1 五大类核心条款**：
   - 经济类（估值 / 股权比例 / 清算优先权）
   - 控制类（董事会席位 / 保护性条款 / 投票权）
   - 反稀释（加权平均 vs full ratchet）
   - 退出类（回购 / 拖售 / 优先清算）
   - 个人类（创始人连带责任 / 竞业）
5. **§10.2 创始人个人责任** — 尽量限制，个人连带 = 不可逆
6. **§10.3 一票否决** — 范围必须精确限定，不能泛化
7. **§10.7 回购对赌** ⭐ — 2021-2024 回购潮前因后果，避险指南
   - **硬回购 + 创始人个人连带是死刑**
   - 接受公司回购但争取 drop-away / basket 条款
   - 对赌业绩 vs 上市 vs 估值调整，本质完全不同
8. **§11 出海架构** — 见 references/index/02-incorporation.md 和 appendix/entity-matrix.md

### YC Post-Money SAFE 2018

- **作者/来源：** Y Combinator
- **发布：** 2018（最新版）
- **URL：** https://www.ycombinator.com/documents
- **License：** CC0-like（YC 明确鼓励全球使用）
- **适用阶段：** Stage 6（种子/天使轮）
- **一句话价值：** 全球种子轮投资协议事实标准
- **核心要点：**
  1. Post-Money 版本（不是早期的 Pre-Money）— 对创始人更有利的稀释计算
  2. Valuation Cap vs Discount 两种机制（或组合）
  3. 无到期日 / 无利息（vs 可转债）
  4. MFN（Most Favored Nation）条款
- **v0.2 计划：** 内置全文到 `bundled/yc-safe-postmoney.pdf`

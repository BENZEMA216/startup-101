## 如何用本 index

| 场景 | 推荐 |
|---|---|
| 准备 pitch 某个基金 | 先读该基金公众号 / 官网最近 3 个月观点 |
| 判断赛道共识 | 对比 a16z / Sequoia / 朱啸虎三方 |
| 想读深度宏观 | 李丰 / Marc Andreessen |
| 想知道同行怎么想 | 锦秋 / 戴雨森 / 石头 |
| 创业迷茫 | Paul Graham / Sam Altman |

## 本 index 在 skill 中的角色

- Stage 1 决策框架 — PG / Sam Altman
- Stage 6 BP 准备 — Sequoia 模板 + 目标基金合伙人公开观点
- Stage 6 投资人名单 — 本 index 是中英 AI VC 快速索引
- 任何阶段的"迷茫时刻"— 读博客比开会有用

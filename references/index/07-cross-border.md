## Tier 1

### EU AI Act — 官方文本与实施时间线

- **作者/来源：** European Commission
- **正式文本：** Regulation (EU) 2024/1689（2024-07-12 公布，分阶段生效）
- **URL：**
  - 官方：https://digital-strategy.ec.europa.eu/en/policies/regulatory-framework-ai
  - 实施时间线：https://artificialintelligenceact.eu/implementation-timeline/
  - GPAI Guidelines：https://digital-strategy.ec.europa.eu/en/policies/guidelines-gpai-providers
  - Code of Practice 介绍：https://artificialintelligenceact.eu/introduction-to-code-of-practice/
- **关键时间锚：**
  - 2025-02-02：禁止性 AI 实践生效
  - **2025-08-02：GPAI Provider 义务生效**
  - **2026-08-02：Art. 50 透明度 + 高风险义务生效**
  - 2027-08-02：Annex II 产品嵌入 AI 合规完成
- **核心内容：**
  - 风险分级（Unacceptable / High / Limited / Minimal）
  - GPAI Provider 五大义务（技术文档 / 训练数据摘要 / 版权政策 / 下游披露 / systemic risk additional）
  - Art. 50 透明度（AI 标识 / 用户告知）
  - Article 27（Art. 27 of GDPR 续）非欧盟公司 EU 代表
- **罚款：** 最高 €35M 或 7% 全球营收（禁止类）/ €15M 或 3%（GPAI）/ €7.5M 或 1.5%（信息错误）

### CAC — 促进和规范数据跨境流动规定（2024-03）

- **作者/来源：** 国家互联网信息办公室
- **发布：** 2024-03-22
- **URL：** http://www.cac.gov.cn/2024-03/22/c_1712776611775634.htm
- **解读：** http://www.cac.gov.cn/2024-03/22/c_1712776625820516.htm
- **一句话价值：** 中国 PIPL 出境新阈值 / 自贸区负面清单机制
- **核心要点：**
  1. **<10 万人 PI（非敏感）+ 无敏感 → 免申报**
  2. **10-100 万人 / <1 万敏感 → SCC 备案 或 认证**
  3. **≥100 万人 / ≥1 万敏感 → 安全评估**
  4. CIIO + 重要数据 → 一律安评
  5. 安评有效期 3 年
  6. 自贸区（上海 / 海南等）可发布负面清单，清单外免评估

### PIPL — 《个人信息保护法》2021

- **发布：** 2021-08-20，2021-11-01 生效
- **URL：** http://www.npc.gov.cn/npc/c2/c30834/202108/t20210820_313088.html
- **核心要点：**
  1. "告知同意"是处理 PI 基础
  2. 敏感 PI（14 岁以下未成年人 / 生物识别 / 宗教信仰 / 医疗健康 / 金融账户 / 行踪）要单独同意
  3. 出境三条路径（安评 / SCC / 认证）
  4. 处罚最高 5000 万 RMB 或上年营收 5%
  5. 域外适用（只要处理中国境内 PI）

### 《网络安全法》修订（2026-01-01 生效）

- **十四届人大常委会第 18 次会议 2025-10-28 通过**
- **URL：**
  - 人大公告：http://www.npc.gov.cn/npc/c2/c30834/202510/t20251028_449076.html
  - 专家解读：https://www.cac.gov.cn/2026-01/02/c_1769093523928606.htm
- **一句话价值：** 首次将 AI 写入国家法律
- **核心变化：**
  1. 新增第 20 条：明确国家支持 AI 基础理论研究 / 算法关键技术 / 训练数据资源 / 算力基础设施 / 伦理规范 / 风险监测评估
  2. 覆盖算法设计 → 数据训练 → 模型部署 → 应用运行全生命周期
  3. 强化法律责任 / 罚款上限提高

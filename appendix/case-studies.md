## 1. Manus / Butterfly Effect

**所在公司：** Butterfly Effect Pte Ltd（Singapore）
**核心产品：** Manus 通用 Agent
**关键节点：** 2025-03 爆红；2025-05 Benchmark 领投 $75M（据 The Information）；估值 $500M 级

### 架构

- **Singapore Pte Ltd** 作主体 — 避 Cayman 复杂 + 贴近 Benchmark
- 创始人团队部分搬至 Singapore，部分仍在国内
- 国内侧一度为 WFOE，后 Singapore 化进程中重组

### 收款 / 融资

- Stripe SG
- 美元基金 Benchmark 领投

### 注意与教训

- **据多方报道，Manus 轮次经历 CFIUS 延期**（中国创始人背景 + AI 关键技术 + US VC 投资 → CFIUS 触发三要素）
- **品牌策略：** "从 Butterfly Effect 到 Manus" 的品牌与主体分离是刻意的海外化
- **在 Stage 1 的启发：** 如果核心目标是 Benchmark / a16z 这类顶级 US VC 的"Global AI"赛道，**Singapore 比 Cayman 更友好 + 比 Delaware 更低 CFIUS 风险**

## 2. HeyGen

**核心产品：** 数字人 / 视频生成
**关键节点：** 2023 从深圳搬 Los Angeles；2024 ARR $35M；Benchmark + Conviction + Sequoia 投资

### 架构

- **Delaware C-corp** 单主体
- 创始人搬至 US（持 O-1 或绿卡路径）
- 训练 / 研发搬 US

### 教训

- **彻底搬家路径**：需要创始人个人身份迁移、数据迁移、模型 IP 搬运（训练日志 / weights / 合同统统迁移）
- 早期曾有 VIE 架构思路，中途放弃转 Delaware 纯海外
- **2024 后几乎所有以视频 / 图像为核心的出海 AI，倾向 HeyGen 模式**（彻底 Delaware）— 降低 PIPL + EU AI Act GPAI 层面风险

### 启发

- 训练数据 / 用户数据从一开始就留海外，**避免后续 PIPL 出境合规难题**
- QSBS 可得（5 年 + $50M 资产门槛）

## 3. Opus Clip

**核心产品：** AI 长视频剪短 C 端工具
**关键节点：** 2023 YC + a16z 种子；2024 A 轮

### 架构

- **Delaware C-corp** 单主体
- 创始人（young，纯技术背景）走 YC 路径

### 教训

- **YC 路径优点：** Delaware 注册 + SAFE 模板 + Stripe Atlas 一站式
- **YC 路径陷阱（创始人税籍）：** 如果仍是中国税务居民 → 退出时中国 20% 个税 + 美国资本利得税可能叠加
- **建议：** 进 YC 前就处理创始人身份（O-1 → 绿卡路径常见）

## 4. Pika Labs

**核心产品：** 文生视频
**关键节点：** 2023 Stanford 背景；Lightspeed / Spark 投资；2024 估值 $500M+

### 架构

- **Delaware C-corp**
- 创始人斯坦福 CS PhD 背景，美国身份

### 教训

- **纯 Archetype C**：创始人已在美 → 无 37 号文 / ODI / WFOE 痛点
- QSBS 优势最大化（持股 5 年 + 公司资产 ≤$50M 退出）
- **训练数据合规**：采用 Shutterstock 等授权数据 + 公开数据 + 合成数据，未来 AI Act 训练数据摘要披露压力较低

### 启发

- 学术背景 + 美国身份的创始人：Delaware 单主体永远是最优
- 不要强行为了"中国 AI"叙事搭红筹

## 5. Dreamina 海外版

**核心产品：** 字节跳动旗下 AI 绘画 / 视频（国内叫"即梦"，海外以"Dreamina"）
**关键节点：** 2024 全球发布；走 TikTok 同款合规通道

### 架构

- ByteDance Cayman 顶层下，海外产品经 **Singapore 实体** 出海
- 与国内主体切割数据流：海外版不回传国内
- 训练主要依赖国内 WFOE + 海外合作数据

### 教训

- **大厂通道优势：** 品牌切割 + 数据切割 + 合规切割三板斧
- 小创业者难以复制，但可以学：**国内 + 海外版本数据 & 模型权重分离**，避免跨境合规
- 海外版采用独立品牌（"Dreamina" vs "即梦"），避免国内品牌风险传导

### 启发

- 纯内资公司也可"被动出海"— 通过海外合作伙伴 / 独立海外子品牌
- 但：创业者不是大厂，**数据隔离成本** 对小团队难以承受。**仍建议 Archetype B**（Cayman/SG 顶层）作为长期架构。

## 6. DeepSeek 海外触达

**核心公司：** 杭州深度求索（关联私募幻方量化）
**核心产品：** DeepSeek V3 / R1 等开源大模型
**关键节点：** 2025-01 R1 发布全球爆火；纯内资架构

### 架构

- **纯内资有限公司**（Archetype A 变体）
- 模型开源（MIT）→ 海外用户直接用开源权重 / 第三方 API
- 官方 chat.deepseek.com 有海外访问但无美元收款主体

### 教训

- **开源模型作为出海手段**（不是出海，是"全球都能用"）
- 纯内资 + 开源 → **没有 PIPL 出境 / EU AI Act GPAI 难题**（用户自己部署自己合规）
- 商业变现靠国内 API / 私有部署 / 战略合作

### 启发

- **如果你做开源基础模型**：纯内资可行，靠开源声誉传播
- 但 C 端订阅 / API 海外变现 → 仍需 Archetype B/C 架构
- R1 发布后 OpenAI / Anthropic 对中国创业 AI 友好度 **整体下降**（赛道敏感度上升），这影响后来者的 CFIUS 风险

## 7. MiniMax / 海螺 / Talkie

**所在公司：** 上海稀宇科技有限公司（境内） + MiniMax Cayman 顶层
**创始人：** 闫俊杰（前商汤副总裁，多模态方向）、杨斌、周昱冲
**核心产品：** 海螺 AI（国内）、Hailuo Video、Talkie（海外陪伴类 AI）、MiniMax 开放平台
**关键节点：** 2024-03 阿里领投 $600M at $2.5B；2026-01 港交所 IPO 估值约 $6.5B

### 架构

- **Cayman 顶层** + 上海稀宇 WFOE + 海外 SPV 承载 Talkie
- 投资方中混合中美（Alibaba / Tencent / MiHoYo / Hillhouse / HongShan / IDG）— Archetype 严格意义上是 "混合"
- 港股 IPO 路径（而非美股）— 避开 CFIUS / PCAOB 长期不确定性，同时保留美元融资的灵活性

### 出海策略

- **Talkie** 在海外 200+ 国家提供 AI 角色陪伴（对标 Character.AI）—— 独立品牌、独立数据
- **海螺 / Hailuo Video** 国内 + 海外双轨（海外品牌 Hailuo AI）
- 开放平台 / API 双市场

### 关键教训

- **"多产品 + 多主体 + 多品牌"** 是中大型 AI 公司应对中国 / 海外合规差异的标配
- 港股 IPO 给中国 AI 公司一条"不完全出海但有外币融资"的路径
- 投资方大量中国互联网大厂 → 估值支撑但 US VC 参与度相对少

### 可提取决策建议

- 产品明确国内 / 海外分线时，**品牌 + 数据 + 模型权重三层切割**
- 若融资方包含大量中国大厂 → 港股 IPO 比美股更可行
- 若要让 Talkie 类产品真正进入美国市场 → 关注 App Store "Chinese-owned" 标签 + 州级法律（CA / TX 未成年人保护）

## 8. Moonshot / Kimi 月之暗面

**创始人：** 杨植麟（清华 → CMU 博士，前 Meta AI / Recurrent.ai）
**核心产品：** Kimi Chat（长上下文王牌，国内）+ Kimi K1/K1.5 推理模型 + 海外版 Kimi
**关键节点：** 2024-02 阿里领投 $1B at $2.5B（36% 股权）；2025 Q3 Series C $500M at $4.3B；2026-02 据报 $700M at $10B

### 架构

- 北京月之暗面科技有限公司（内资）+ 可能存在海外 SPV（未公开披露完整架构）
- 投资方高度依赖中国大厂：**Alibaba、Tencent、Meituan 王慧文、IDG、5Y Capital**
- 美元基金参与度低（阿里持股接近 36%→ 实质准子公司）

### 出海策略

- Kimi 主战场仍是国内 C 端 + 开发者平台
- 2025 下半年海外收入明显上升（SCMP 报道）
- 但 **CFIUS 风险 + 阿里背景**使 US VC 参与低

### 关键教训

- **"阿里系"大厂战投 ≥ 30% 股权** 实质构成关联方 — 未来想独立 IPO 需特别结构设计
- AI 大模型公司国内"烧钱王"模式：估值靠大厂资金 + 模型榜单 + 产品用户
- **警示：如果你单家大厂持股超 20% 就难以再融美元**

### 可提取决策建议

- 大厂战投 20-25% 是上限红线，超过即变成"实质控股"
- Series C 前就锁定外汇路径（港交所 / 美股）；后期重组成本极高
- 海外收入做起来但合规准备不足 → PIPL / EU AI Act 爆雷潜在

## 9. 百川智能 Baichuan AI

**创始人：** 王小川（搜狗创始人，连续创业者）
**核心产品：** Baichuan 系列大模型（开源 + 商业）、医疗 AI 方向
**关键节点：** 2023-10 $300M A 轮 at $1B（估值独角兽）；2024-07 $700M 轮 at $2.8B；**2024 年下半年战略聚焦医疗 AI（B2B）**

### 架构

- **纯内资有限公司**（北京百川智能科技）
- 投资方：**人民币基金为主**（阿里、腾讯、小米、深创投、中金启辰等）
- 没有 Cayman 顶层 / 没有美元融资路径

### 战略转型

- 2023-2024 上半年：通用大模型竞赛，对标 DeepSeek / Moonshot
- 2024 下半年起明确**聚焦医疗 AI** — 王小川本人深度参与医疗垂直方向
- 开源 + B2B 商业双轨：Baichuan-M1 / Baichuan4-Medical

### 关键教训

- **连续创业者的优势 + 路径依赖**：王小川搜狗退出后直接做 AI，能快速拉人民币基金
- **通用大模型 → 垂直赛道**的转型是 2024 后中国 AI 独角兽的共同选择（资源集中）
- 纯内资架构 + 人民币融资 + 医疗赛道 = **典型 Archetype A "重行业"路径**

### 可提取决策建议

- Archetype A 要想做大，**聚焦一个垂直**比通用竞赛更可行
- 医疗 / 金融 / 法律等重监管行业内资公司有天然优势（外资准入限制）
- 创始人个人 IP（王小川搜狗背景）对人民币基金的拉动力远强于技术参数

## 10. 智谱 AI / Z.ai

**创始人：** 清华 KEG 实验室孵化（张鹏 CEO）
**核心产品：** GLM-4 系列、ChatGLM 开源、智能体平台 AutoGLM
**关键节点：** 2023-2024 多轮 RMB 融资估值 >$3B；**2025-01-15 美国 BIS Entity List（首家被制裁的中国大模型公司）**；2025-07 国际品牌改名 **Z.ai**，发布 GLM-4.5；2026-01 **HKEX IPO（改名 Knowledge Atlas 上市）**

### 架构

- 北京智谱华章科技（内资）+ 清华 THU 创投背景
- 2025 被 Entity List 后，**5 周内设立浙江智谱新篇科技（未列入 Entity List 的子公司）** — 典型"BIS 50% rule"子公司隔离动作
- 2026 香港 IPO 更名 "Knowledge Atlas Technology JSC Ltd"

### 制裁应对

- **Footnote 4 designation**：不仅限制美国原产物项，还限制**含有美国技术 / 软件 / 组件的外国产物**（最严一档）
- 官方立场："没依赖美国大模型技术，不受实质影响" — 实际上仍有算力、EDA、网络硬件链条影响
- 国际品牌切割：Zhipu → Z.ai（对标 X.ai 叙事）

### 关键教训

- **"THU 背景 + 国家 AI 叙事"** 是双刃剑：国内融资顺 + 海外敏感度高
- 被制裁不是终点 — 港股 IPO 路径 + 子公司隔离仍可走
- **v0.3 启示：如果你的产品会被贴"China AI"标签 + 有 THU / CAS / 国企关联 → 美国资本路径本身就要重新考虑**

### 可提取决策建议

- 若创始团队有强政府 / 院校关联，**提前规划子公司 + 品牌隔离**（不要等制裁后才做）
- 港股 IPO 正在成为"不能去美股的中国 AI 公司"的退出路径
- 关注 BIS 50% rule：**未被直接制裁的子公司目前仍可采购美国技术**（但监管正在收紧）

## 11. 01.AI / 零一万物

**创始人：** 李开复（前 Google China / 创新工场）
**核心产品：** Yi 系列模型（开源 + 商业）、Yi-Lightning
**关键节点：** 2023 创立，多轮美元融资估值 >$1B；**2024 Q4 战略收缩：预训练团队转阿里通义 + infra 转阿里云 + 成立联合实验室**；2025 起聚焦 B2B 行业小模型

### 架构

- **Cayman 顶层 + 境内 WFOE**（Archetype B 典型结构）
- 美元 + 人民币混合投资方（阿里、创新工场等）
- 2024-12 起与阿里云成立"产业大模型联合实验室" — 实质上**部分核心团队 + infra 转入阿里**

### 战略收缩

- **预训练大军 → 阿里通义**：放弃继续训练超大基模
- **Infra 团队 → 阿里云**：算力 / 服务能力整合到阿里云商业体系
- 01.AI 保留品牌 + 部分团队，聚焦 **B2B 行业模型** 和 **小参数场景模型**
- Kai-Fu Lee 公开否认"整体卖身"，但外界普遍解读为"体面收缩"

### 关键教训

- **2024 AI 泡沫回调的样板**：明星创始人 + 大额美元融资 + 没做出爆款 = 资源整合到大厂
- **美元融资 + Cayman 架构不能阻止"整合进国内大厂"的结果**：资本路径与最终技术去向脱钩
- "六小虎"（智谱 / Moonshot / 百川 / MiniMax / 01.AI / Stepfun）中 01.AI 是第一个明确收缩的

### 可提取决策建议

- **AI 基模创业的窗口已关**（2026 视角）：没有 DeepSeek / OpenAI 级别的模型能力就不要做通用基模
- 美元融资 + 海外架构**不能保证独立性** — 如果产品烧钱快且没爆款，最终归属仍是大厂
- 如果你是"六小虎后 AI 创业"的创始人：**别再瞄准通用基模赛道**，从垂直行业 / 应用 / Agent 开始

## 跨案例规律

| 规律 | 说明 |
|---|---|
| **2024 后新出海默认 Delaware 或 Singapore** | VIE / Cayman+VIE 几乎消失（除非特定牌照行业） |
| **视频 / 图像 / 数字人类产品尤其倾向彻底搬家** | 训练数据 + 用户数据的 PIPL 压力最大 |
| **创始人身份是硬约束** | 身份没迁移，架构迁移效果减半（FBAR / 全球征税 / 37 号文） |
| **CFIUS 是 US VC 路径的常客** | 中国创始人 + AI + US 资本三要素 → 提前预审 |
| **品牌切割** | 大厂案例共性：国内品牌 ≠ 海外品牌（避免舆论传染） |
| **开源作为出海路径** | DeepSeek / Qwen 的模式值得研究，但不适合 C 端订阅 |

| **Entity List 预案必须前置** | Z.ai 案例：被列入后 5 周内设立子公司。具备 THU/CAS/国企关联的创业者应在融资前就规划 |
| **2024 后通用基模窗口关闭** | 01.AI 收缩样板；后续创业者应从垂直 / 应用 / Agent 切入 |

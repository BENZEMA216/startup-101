# AI Filing Guide — 算法备案 / 大模型备案 / 登记三轨详解

> last_policy_review: 2026-04-15
> 用于 Stage 5（AI 产品上线前）。

## 一、三轨总览

| 轨道 | 法定依据 | 触发条件 | 主管 | 周期 | 典型成本 |
|---|---|---|---|---|---|
| **A. 算法备案**（生成合成 / 推荐 / 其他具舆论属性算法） | 《互联网信息服务算法推荐管理规定》2022 + 《生成式人工智能服务管理暂行办法》2023 | 算法面向中国公众 + 具舆论属性或社会动员能力 | 国家/省网信办 | 1-3 个月 | 2-5 万 RMB（服务商） |
| **B. 大模型备案** | 《生成式人工智能服务管理暂行办法》2023 + 《生成式人工智能服务安全基本要求》TC260-003 | **自研**生成式模型 + 面向中国公众 | 国家/省网信办 + 专家评审 | **3 个月+** | 30-80 万 RMB |
| **C. 大模型登记** | 同 B | **调用已备案模型 API** 提供面向公众服务 | 省级网信办 | 2-6 周 | 0.5-3 万 RMB |

> **双备案原则：** 自研生成模型通常触发 **生成合成算法备案 + 大模型备案** 两项，缺一不可。

---

## 二、决策树（profile 驱动）

```
自研模型？
├─ 是 → 面向中国公众？
│       ├─ 是 → 生成内容？
│       │       ├─ 是 → 走 [B] 大模型备案 + [A] 生成合成算法备案
│       │       └─ 否 → 走 [A] 算法备案（推荐 / 其他）
│       └─ 否 → 仅科研 / 内部 → 免备案（但 ToB 严肃应用仍建议登记一轨）
└─ 否（调 API）→ 面向中国公众？
        ├─ 是 → 走 [C] 大模型登记
        └─ 否（ToB / 海外） → 免备案
```

> **"舆论属性或社会动员能力" 实操判定：** 面向公众 + 具传播性 + 有内容输出 ≈ 命中。实际只要 app / 网站 / 小程序对 C 端开放 + AI 能说话，默认命中。

## 三、[A] 算法备案 详解

### 适用算法类型（6 类）

1. **生成合成类**（文/图/音/视频 AI 生成）
2. **个性化推送类**（推荐系统）
3. **排序精选类**
4. **检索过滤类**
5. **调度决策类**（打车 / 派单）
6. **其他具舆论属性**

### 材料清单

- [ ] 主体资质：营业执照（**经营范围含"技术开发 / 人工智能"等**）
- [ ] 算法安全自评估报告（重点：违法不良信息防范 / 数据安全 / 用户权益）
- [ ] 算法基本情况表（算法名称 / 类型 / 应用场景 / 数据来源 / 模型架构简述）
- [ ] 拟公示内容（算法功能 / 目的 / 机制机理）
- [ ] 产品 demo 视频 / 链接
- [ ] 安全负责人身份证明 + 联系方式

### 流程

1. **互联网信息服务算法备案系统**（https://beian.cac.gov.cn/）填报
2. 省级网信办初审（10-20 工作日）
3. 国家网信办复核 → 批准 → **双随机抽查**
4. 公示于 https://beian.cac.gov.cn 查询页
5. **重大变更重新备案**（算法机制重大变化 / 应用场景扩展 / 主体变更）

### 时间

- 首次：1-3 个月
- 补正：+1 个月

### 红旗

- 🚩 营业执照经营范围漏 AI 类目 → 备案主体不适格（回到 Stage 2 改经营范围）
- 🚩 自评估报告模板化 / 不覆盖违法不良信息 → 打回

## 四、[B] 大模型备案 详解（最重）

### 适用

**自研基础模型 / 垂直大模型 + 面向中国公众生成式服务**。

> 典型：Qwen / DeepSeek / Doubao / Moonshot 等自研模型对外服务；某垂直医疗 / 法律大模型对 C 端。

### 核心合规要求（TC260-003）

| 维度 | 2026 最新要求 |
|---|---|
| **语料合规** | 境外训练语料占比 **≤30%**；每个语料源须有完整授权证明；不得含淫秽 / 暴恐 / 违法不良信息 |
| **拒答率** | 敏感问题拒答率 **≥95%** |
| **非拒答测试** | 合格率 **≥96%**（不该拒的别拒） |
| **内容标识** | **显式水印 + 隐式哈希 双标识**（2026 新增强制） |
| **个人信息** | 训练不得未经授权使用他人 PI；同意机制必须明确 |
| **安全机制** | 关键词过滤库 + 敏感话题检测模型 + 人工审核兜底 |

### 材料清单（约 20 份）

- [ ] 企业主体材料（执照 / 章程 / 法人证明）
- [ ] **语料标注规则**
- [ ] **语料合法性说明**（含每个语料源授权）
- [ ] **安全性评估报告**（第三方机构）
- [ ] **模型服务协议 + 用户协议**
- [ ] **关键词库 + 敏感话题样本库**
- [ ] **人工审核机制说明**
- [ ] **投诉举报渠道说明**
- [ ] **违规处置机制**
- [ ] **应急预案**
- [ ] **安全负责人任命 + 身份证明**
- [ ] **算法备案回执**（生成合成类先备，再大模型备）
- [ ] **模型说明文档**（参数量 / 训练算力 / 训练数据规模 / 能力评估）
- [ ] **红队测试报告**
- [ ] **等保 2.0 定级备案**（>=3 级常见）
- [ ] **个人信息保护影响评估（PIIA）**

### 流程

1. **生成合成算法备案**（先走 [A] 轨）→ 拿回执
2. 提交**大模型备案系统**
3. 省级网信办受理 → 要求**专家评审**（高风险级别）
4. 材料补正（通常 1-3 轮）
5. 国家网信办复核 → 批准
6. 入库 CAC 公示列表

### 时间 / 成本

- **通常 3-6 个月**（含补正）
- **30-80 万 RMB**（顾问 + 第三方安评 + 人工审核配置）
- 紧急 "绿色通道" 不存在 — 排队是排队

### 推荐服务商类型

- **综合顾问：** 中伦网络 / 锦天城 / 观韬中茂 / 安杰世泽（律所路径）
- **内容安全 + 备案一体化：** 数美 / 易盾 / 阿里云盾 / 腾讯云天御（厂商路径）
- **第三方安评机构：** 中国信通院 / 中国电子技术标准化研究院

（**非背书**，仅示意。）

### 红旗

- 🚩 **境外语料 >30%** → 打回（这是最常见的卡点）
- 🚩 **未做显式 + 隐式双水印** → 2026 新规硬要求
- 🚩 **语料授权证明缺失** → 一刀切打回
- 🚩 **拒答率测试未达 95%** → 补正至符合

## 五、[C] 大模型登记 详解

### 适用

**不自研基础模型**，通过 API / SDK 调用已备案大模型（如调 Qwen / DeepSeek / 豆包 API）+ 面向中国公众提供生成式服务。

### 材料（相对轻）

- [ ] 企业主体材料
- [ ] 应用服务说明
- [ ] **所调用模型的备案号**
- [ ] 内容过滤 / 人工审核机制
- [ ] 用户协议 / 隐私政策
- [ ] 违规处置机制
- [ ] 投诉举报渠道

### 流程

1. 在 https://beian.cac.gov.cn 填报"登记"（非"备案"）
2. 省级网信办受理
3. 批准 → 发放登记号

### 时间 / 成本

- 2-6 周
- 0.5-3 万 RMB（服务商或自行）

### 注意

- **登记不等于免责**：下游应用仍需自行做内容过滤 + 审核 + 投诉响应
- **模型切换必须重新登记**（从 Qwen 切到 DeepSeek → 重来）

## 六、深度合成算法独立备案

即使走了 [A] 生成合成类备案，如果产品涉及：
- 人脸合成 / 换脸
- 语音合成 / 克隆
- 数字人 / 虚拟形象
- 图文 / 视频内容强编辑

需按《互联网信息服务深度合成管理规定》额外做**深度合成算法备案**，并在生成内容显著位置添加"AI 合成 / 深度合成"显式标识。

**强制标识：** 显式（肉眼可见）+ 隐式（元数据/水印）双通道，2026 起 CAC 抽查严。

## 七、时间规划建议

| 规划目标 | 建议提前量 |
|---|---|
| 仅做登记（C 轨） | 上线前 **2 个月** |
| 算法备案（A 轨） | 上线前 **3 个月** |
| 大模型备案（B 轨） | 上线前 **6 个月**（含 2 轮补正） |
| B + A 双备 | 上线前 **6-9 个月** |

## 八、跨境场景叠加

如果你同时要在国内 + 海外上线：
- **国内** → 本文三轨
- **欧盟** → EU AI Act GPAI（2025-08 起，自研）/ Art. 50（2026-08 起，所有 AI 应用）
- **训练数据** → PIPL 出境（见 compliance-matrix.md）

**冲突点：** 国内大模型备案不鼓励披露训练数据细节；EU AI Act 要求公开训练数据摘要。**建议采取"语义粒度差异化披露"策略 — 国内备案走内部 detailed，EU 摘要走"数据类别 + 粗占比"。具体措辞律师审阅。**
